The dataset was imported from an Excel file into R and initially inspected to understand its structure, variable types, and overall composition. A detailed missing value analysis was conducted to identify and quantify missing data across all variables. During this process, hidden missing values represented by symbols such as “-” were detected and converted into standard NA format to ensure consistency. Variables with extremely high proportions of missing values were removed from the dataset to prevent unreliable analysis. For the remaining data, categorical variables were imputed using the mode, replacing missing entries with the most frequent category, while numerical variables were imputed using the median to maintain robustness against skewness and outliers. After imputation, the dataset was re-evaluated to confirm that missing values had been effectively handled. A comparison of missing values before and after preprocessing was performed to assess improvements in data completeness. Additionally, data distributions were examined using graphical methods to understand variable behavior and identify any irregularities. Finally, the cleaned dataset was exported into Excel and CSV formats, making it ready for further statistical and multivariate analysis.
